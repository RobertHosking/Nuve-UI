# Nuve UI

# Getting Started

### Prerequsites:
- npm
- nodejs
- sass

`cd Nuve-UI`

`npm install`

`npm start`

Navigate to `localhost:3000`

To start the automatic Sass compiler in a seperate terminal:

`sass --watch styles/sass/style.sass:styles/css/style.css`
